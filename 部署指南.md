## 本地运行测试

### 1. 安装 Python

确保已安装 Python 3.8 或更高版本。

### 2. 安装依赖

在项目目录下打开命令行，执行：

- 管理后台：http://localhost:5001/admin

## 服务器部署（火山引擎）

### 方式一：使用宝塔面板部署（推荐）

#### 步骤 1：安装宝塔面板

通过 SSH 连接到服务器，执行以下命令：

**CentOS 系统：**
```bash
yum install -y wget &amp;&amp; wget -O install.sh https://download.bt.cn/install/install_6.0.sh &amp;&amp; sh install.sh ed8484bec
```

**Ubuntu/Debian 系统：**
```bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh &amp;&amp; sudo bash install.sh ed8484bec
```

安装完成后，记录显示的：
- 外网面板地址
- 用户名
- 密码

#### 步骤 2：登录宝塔面板

在浏览器中打开外网面板地址，使用用户名和密码登录。

#### 步骤 3：安装 Python 环境

1. 在宝塔面板左侧菜单点击「软件商店」
2. 搜索「Python」
3. 安装 Python 3.8 或更高版本

#### 步骤 4：上传网站文件

1. 在宝塔面板左侧菜单点击「文件」
2. 进入 `/www/wwwroot/` 目录
3. 创建新文件夹 `miheai`
4. 进入 `miheai` 文件夹
5. 上传以下文件：
   - app.py
   - index.html
   - requirements.txt
   - templates/ 文件夹（包含 login.html 和 admin.html）

#### 步骤 5：安装项目依赖

1. 在宝塔面板左侧菜单点击「终端」
2. 执行以下命令：
```bash
cd /www/wwwroot/miheai
pip install -r requirements.txt
```

#### 步骤 6：配置 Python 项目

1. 在宝塔面板左侧菜单点击「网站」
2. 点击「添加站点」
3. 填写信息：
   - 域名：填写您的域名（或服务器 IP）
   - 数据库：不创建
   - PHP版本：纯静态
4. 点击「提交」

5. 点击刚创建的网站右侧的「设置」
6. 点击「配置文件」
7. 添加以下配置（替换原有内容）：

```nginx
server {
    listen 80;
    server_name 您的域名或IP;
    
    location / {
        proxy_pass http://127.0.0.1:5001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

8. 点击「保存」

#### 步骤 7：使用 Supervisor 管理 Flask 应用

1. 在宝塔面板左侧菜单点击「软件商店」
2. 搜索「Supervisor管理器」并安装
3. 点击「设置」
4. 点击「添加守护进程」
5. 填写信息：
   - 名称：miheai
   - 用户：root
   - 运行目录：/www/wwwroot/miheai
   - 启动命令：python app.py
6. 点击「确定」

#### 步骤 8：访问网站

在浏览器中访问您的域名或服务器 IP。

### 方式二：使用 Gunicorn + Nginx 部署

#### 1. 安装 Gunicorn

```bash
pip install gunicorn
```

#### 2. 测试运行

```bash
cd /www/wwwroot/miheai
gunicorn -w 4 -b 127.0.0.1:5001 app:app
```

#### 3. 创建 Systemd 服务

创建文件 `/etc/systemd/system/miheai.service`：

```ini
[Unit]
Description=米核AI网站
After=network.target

[Service]
User=root
WorkingDirectory=/www/wwwroot/miheai
Environment="PATH=/usr/local/bin"
ExecStart=/usr/local/bin/gunicorn -w 4 -b 127.0.0.1:5001 app:app
Restart=always

[Install]
WantedBy=multi-user.target
```

#### 4. 启动服务

```bash
systemctl daemon-reload
systemctl start miheai
systemctl enable miheai
```

#### 5. 配置 Nginx

参考方式一的步骤 6。

## 注意事项

1. **安全建议**：
   - 首次登录后立即修改默认密码
   - 定期备份数据库文件 `miheai.db`
   - 配置防火墙，只开放必要端口

2. **数据库备份**：
   - 数据库文件为 `miheai.db`
   - 定期复制该文件进行备份

3. **上传文件**：
   - 上传的图片保存在 `uploads/` 文件夹
   - 确保该文件夹有写入权限

## 常见问题

### Q: 端口 5001 被占用怎么办？

A: 修改 `app.py` 最后一行的端口号：
```python
app.run(host='0.0.0.0', port=8000, debug=True)
```

### Q: 如何重启应用？

A: 
- 使用宝塔面板：在 Supervisor 管理器中重启
- 使用 Systemd：`systemctl restart miheai`

### Q: 数据库文件在哪里？

A: 与 `app.py` 在同一目录下，文件名为 `miheai.db`
